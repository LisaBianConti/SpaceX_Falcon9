# IBM Data Science Capstone Project - SpaceX Falcon 9 Landing Prediction

## Background

Launching rockets hold high risks and high costs for stakeholders. 
Several companies in space launching industry have developped innovative approaches to guarantee launches and to reduce costs.
SpaceX offers the launch of Falcon 9 rockets at 62 million USD while other competitors demand at least 165 million USD. 
The technology behind the appealing economic launch is that SpaceX reduces cost by recycling the first stage of the launch. 
Hence, the key question is whether SpaceX can keep this promise and therefore reduce the cost for stakeholders, or is it an ostentatious marketing strategy. 
To answer this question with a verifiable confidence, we need to obtain the successful rate of the SpaceX first stage landing. 

## Objectives

The main purpose of this project is to predict if the first stage of SpaceX Falcon 9 rockets will land successfully using machine learning techniques.
The forecast model is useful for projecting the cost effectiveness of SpaceX rocket launches for stakeholders in the space exploration industry.
In addition, the results can be advantageous for competitors that wishes to outperform SpaceX in rocket launch.

## Results

The data shows that the success rate has a increasing trench during 2013 to 2020. 
Certain orbits showed more promising results, among which the most successful launch location is KSC LC-39A.
In terms of machine learning models, decision tree classifier stands out to be the most reliable for the relevant dataset.

## Methods

- Data collection via REST API, SQL and web scrapping
- Data wrangling and analysis

- Machine learning prediction
